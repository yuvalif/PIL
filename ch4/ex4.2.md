Preferable format added as an optional output to the `xxd` binary to hex tool.
See [this PR](https://github.com/vim/vim/pull/15686).

For example, if we have the file `test`:
```
hello world
hello world
hello world
hello world
hello world
hello world
hello world
hello world
hello world
hello world
hello world
hello world
hello world
hello world
hello world
hello world
```
